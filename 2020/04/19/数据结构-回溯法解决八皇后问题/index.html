数据结构-回溯法解决八皇后问题

发表于2020-04-19更新于2021-12-23字数统计1.2k阅读次数

回溯法：

从问题的某一种可能出发, 搜索从这种情况出发所能达到的所有可能, 当这一条路走到” 尽头 “的时候, 再倒回出发点, 从另一个可能出发, 继续搜索. 这种不断” 回溯 “寻找解的方法, 称作” 回溯法 “。
回溯是一种算法思想，可以用递归实现。

八皇后问题

八皇后问题，是一个古老而著名的问题，是回溯算法的典型案例。该问题是国际西洋棋棋手马克斯·贝瑟尔于 1848 年提出：在 8×8 格的国际象棋上摆放八个皇后，使其不能互相攻击，即：任意两个皇后都不能处于同一行、 同一列或同一斜线上，问有多少种摆法(92)。

代码说话:

public class Queue8 {


//八皇后问题

//8行8列的格子，每一行放一个皇后，任意两行的皇后不能在同一列，同一斜线

//回溯法
/*
思路：第一行第一列放上皇后，

接着第二行从第一列开始放皇后，判断皇后是否符合要求，直到找到合适的第一个位置。

其他行一样，每次放一个位置都会比较当前行和已经放过皇后的行。

实践证明，当你按照这样放的时候，到了某一行，你会发现

这一行找完所有位置没有一个是符合要求的，那么重点来了，

就回溯到前一行，从你放皇后的位置向右挪一位,找不到合适的位置就再挪，直到循环结束，还是不行就回到再前一行

直到找到你这行的第二个与下面的其他放过皇后的行不冲突的位置，

这一行找到位置了，

然后就进到到原来没有位置那一行（就是这一行的后一行），

再次寻找合适的位置。以此类推，当有一行的位置都不符合，就回到前一行，找到合适位置，

如果前一行还是没有合适的位置，就回到再前一行。。

假如找到位置了就递增一行。

就这样，一定会找到的一个正确的位置，当你的行数到达第8行并且已经成功后，会退出方法，

那么你第八行的剩下的循环位置会继续执行，

它会向右把皇后加一个位置，这个时候会发生冲突了，那么就持续推进，直到列数走完，

走完了就回退了前一行，前一行的皇后再向前移动，直到找到合适的位置再进入后一行，后一行再找位置，

再不行再回退，前一行不行就再回退，这样的话新一轮的解法求解又即将出来了。。。。。。

当最外层的循环执行完了，一切结束。
*/


int max=8;//8个皇后
int [] arr=new int [max];
static  int count=0;
static  int judegCount=0;
 public static void main(String args[]) {

 //我们用一个数组来存放结果
     //如：array[8]={0,1,2,3,4,5,6,7,8}
       //代表8个皇后，arr[i]=val 代表第（i+1）个皇后放在地(i+1)行的第（var+1）列

     Queue8 queue8=new Queue8();
     queue8.check(0);
     System.out.println("共"+count+"种解法");
     System.out.println("共"+judegCount+"种突出检测");

 }






//放皇后的方法
public void check(int n){


     //加个条件，如果第8行加入皇后不发生冲突了，就停止进入第九行，因为没有！
    if(n==max){


        //这个时候就可以打印一种解法的结果了

        print();
        count++;
        System.out.println();

        return;

    }

     //n从0开始，代表行因为这样容易对结果数组的位置赋值

    for(int i=0;i<max;i++){

        //进来的皇后位置先是该行的第一列
        //i是列
        arr[n]=i;

        //放完后与其他已经放了的行不冲突了就去下一行，否则就继续挪位置(列)
        if(judge(n)){

            //说明不冲突
            //去下一行
            check(n+1);


        }


    }

}

private void print() {

     //打印每一种解法
    for(int i=0;i<arr.length;i++){

        System.out.printf(arr[i]+" ");
    }

}

private  boolean judge(int n){

     //传进来的n指的是当前行，判断当前行的冲突是否与其他的几个n-1行是否冲突
    //判断是否在同一列,是否在同一斜线

judegCount++;

   for(int i=0;i<n;i++){

       if(arr[i]==arr[n]||Math.abs(n-i)==Math.abs(arr[n]-arr[i])){

           //说明是在同一斜线上

           return  false;
       }

   }
   return  true;

}


}
