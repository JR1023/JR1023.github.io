顺序存储二叉树

从数据存储来看，数组存储方式和树的存储方式可以相互转换，即数组可以转换成树，树也可以转换成数组

特点：

  1. 顺序二叉树通常只考虑完全二叉树

  2. 第 n 个元素的左子节点为 2 * n + 1

  3. 第 n 个元素的右子节点为 2 * n + 2

  4. 第 n 个元素的父节点为 (n-1) / 2

  5. n : 表示二叉树中的第几个元素(按 0 开始编号如图所示)

顺序存储二叉树遍历

给你一个数组 {1,2,3,4,5,6,7}，要求以二叉树前，中，后序遍历的方式进行遍历。

代码如下：

 public class ArrBinaryTreeDemo {


//顺序存储二叉树

//树可以转成数组，数组也可以转成树
//如int [] arr={1,2,3,4,5,6},那么从索引0开始，二叉树从上到下，从左到右也是按照数组的索引来排的，对应的也是数组的数据

 public static void main(String args[]) {


     //顺序存储二叉树按照前序，中序，后序遍历

     ArrBinary arrBinary=new ArrBinary(arr);

      arrBinary.preOrderStart();

      //中序
    System.out.println("前序完成");

 }

//顺序存储二叉树类
class ArrBinary{


private  int [] arr;  //需要一个数组

public ArrBinary(int[] arr) {
    this.arr = arr;
}


public  void  preOrderStart(){

    preOrder(0);
}
public  void  inOrderStart(){

    inOrder(0);
}
public  void  postOrderStart(){

    postOrder(0);
}

//前序遍历
public  void preOrder(int index){

    //先输出当前节点
    System.out.println(arr[index]);

    //左边
    if(2 * index+1<arr.length){

        //去左边的节点
        preOrder(2 * index+1);

    }

    //右边
    if(2 * index+2<arr.length){

        //去右边的节点
        preOrder(2 * index+2);
    }

}


//中序遍历

public  void inOrder(int index){

    //先左边

    if(2 * index+1<arr.length){

        inOrder(2 * index+1);
    }

    System.out.println(arr[index]);

    //右边

    if(2 * index + 2<arr.length){

        inOrder(2 * index + 2);

    }

}
//后序遍历

public  void postOrder(int index){

    //先左边

    if(2 * index+1<arr.length){

        postOrder((2 * index+1));
    }
    //右边

    if(2 * index + 2<arr.length){

        postOrder(2 * index + 2);

    }

    System.out.println(arr[index]);
}


}

线索化二叉树

问题：将数列 {1, 3, 6, 8, 10, 14 } 构建成一颗二叉树. n+1=7

  1. 当我们对上面的二叉树进行中序遍历时，数列为 {8, 3, 10, 1, 6, 14 }
  2. 但是 6, 8, 10, 14 这几个节点的 左右指针，并没有完全的利用上.
  3. 如果我们希望充分的利用 各个节点的左右指针， 让各个节点可以指向自己的前后节点,怎么办?
  4. 解决方案-线索二叉树

线索二叉树基本介绍

  1. n 个结点的二叉链表中含有 n+1 【公式 2n-(n-1)=n+1】 个空指针域。利用二叉链表中的空指针域，存放指向 该结点在某种遍历次序下的前驱和后继结点的指针（这种附加的指针称为”线索”）

  2. 这种加上了线索的二叉链表称为线索链表，相应的二叉树称为线索二叉树(Threaded BinaryTree)。根据线索性质 的不同，线索二叉树可分为前序线索二叉树、中序线索二叉树和后序线索二叉树三种

  3. 一个结点的前一个结点，称为前驱结点 4) 一个结点的后一个结点，称为后继结点

线索二叉树应用案例

应用案例说明：将下面的二叉树，进行中序线索二叉树。中序遍历的数列为 {8, 3, 10, 1, 14, 6}

中序遍历的结果：{8, 3, 10, 1, 14, 6}

说明: 当线索化二叉树后，Node 节点的 属性 left 和 right ，有如下情况:

  1. left 指向的是左子树，也可能是指向的前驱节点. 比如 ① 节点 left 指向的左子树, 而 ⑩ 节点的 left 指向的 就是前驱节点.

  2. right 指向的是右子树，也可能是指向后继节点，比如 ① 节点 right 指向的是右子树，而⑩ 节点的 right 指向 的是后继节点.

//线索化二叉树的方法（中序）

public  void   threadedBinaryTreeInOrder(HeroNode2 node){

    if(node==null){
        //节点为空
        return;
    }

    //中序线索化


    //先左边
    threadedBinaryTreeInOrder(node.getLeft());

    //处理当前节点
    //先处理左指针

    if(node.getLeft()==null){

        //左指针为空的情况下，那么就可以指向前驱节点了
        node.setLeft(pre);
        node.setLeftType(1);

    }

    //接下来到右节点,指向后继节点这种操作可以让node移动了一个位置再指向，比如说{8, 3, 10, 1, 14, 6},
    // 当前是8,然后node移动到3的时候，那么你让pre同时也指向node的前一个也就是8,那么pre在指向3就可以了

    if(pre!=null && pre.getRight()==null){

        //前置节点不为空并且其右指针是没有指向的时候就可以操作后继节点了

        pre.setRight(node);
        pre.setRightType(1);
    }

    //让pre指向node,这样从8开始的话，当到3的时候上面的pre就是8，这样8就可以和3连接上了

    pre=node;

    //右边

    threadedBinaryTreeInOrder(node.getRight());



}

    System.out.println(node);

    HeroNode2 resNode=null;   //这个值的设置就是为了纪录你进到一个节点里面是不是你要找的，如果找到了它的值就不是空了，不然就是null;
    if(this.left!=null){

        //如果向左递归找到了就返回,就没必要向右边递归了,所以要提前定义一个变量来接收
        resNode=this.left.preOrderSearch(no);

    }
    if(resNode!=null){

        return  resNode;
    }
    if(this.right!=null){

        resNode=this.right.preOrderSearch(no);

    }

    //这个时候一定要返回了，无论resNode空还是不空,因为要有递归退出条件

    return  resNode;



}

//中序遍历

public  HeroNode2 inOrderSearch(int no){

    HeroNode2 resNode=null;//这个值的设置就是为了纪录你进到一个节点里面是不是你要找的，如果找到了它的值就不是空了，不然就是null;

    //先向左边

    if(this.left!=null){

        resNode=this.left.inOrderSearch(no);
    }
    if(resNode!=null){

        return  resNode;
    }

    System.out.println("进入中序查找");
    if(this.no==no){

        return  this;

    }

    //否则就向右边

    if(this.right!=null){

        resNode=this.right.inOrderSearch(no);

    }

    return  resNode;


}



//后序查找

public  HeroNode2 postOrderSearch(int no){

    //先左边
    HeroNode2 resNode=null;//这个值的设置就是为了纪录你进到一个节点里面是不是你要找的，如果找到了它的值就不是空了，不然就是null;

    //先向左边

    if(this.left!=null){

        resNode=this.left.postOrderSearch(no);
    }
    if(resNode!=null){

        return  resNode;
    }

    if(this.right!=null){

        resNode =this.right.postOrderSearch(no);

    }
    if(resNode!=null){

        return  resNode;
    }

    System.out.println("进入后序查找");
    if(this.no==no){

        return  this;
    }

    return  resNode;
}



//删除

public  void delNo(int no){


 /*
    1. 因为我们的二叉树是单向的，所以我们是判断当前结点的子结点是否需要删除结点，而不能去判断 当前这个结点是不是需要删除结点

    2. 如果当前结点的左子结点不为空，并且左子结点 就是要删除结点，就将 this.left = null; 并且就返回 (结束递归删除)

    3. 如果当前结点的右子结点不为空，并且右子结点 就是要删除结点，就将 this.right= null ;并且就返回 (结束递归删除)

    4. 如果第 2 和第 3 步没有删除结点，那么我们就需要向左子树进行递归删除

    5. 如果第 4 步也没有删除结点，则应当向右子树进行递归删除.

  */

    if(this.left!=null &&this.left.no==no){

        //说明该节点的左节点符合条件
        this.left=null;
        return;

    }
    if(this.right!=null &&this.right.no==no){

        //说明该节点的右节点符合条件
        this.right=null;
        return;

    }

    //否则呢，就要递归向左寻找

    if(this.left!=null){

        this.left.delNo(no);
    }

    //递归向右寻找
    if(this.right!=null){

        this.right.delNo(no);
    }


}
